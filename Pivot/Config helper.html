Character

Characters are any type of Pokémon or NPC that are found on the Characters folder. Defined characters are not necessarily Pokémon you can play as or even Pokémon you can encounter as AI.

Attributes

Below you will find all modifyable attributes for each character, you can press on a specific attribute to jump to a more detailed explanation of that attribute.

Attribute Default value Example Datatype Description
name nil "Murkrow" String The display name of this character
internal nil :MURKROW Symbol The internal name of this character (used for things like cries, announcer and charsets)
melee nil :WINGATTACK Symbol The internal name of this character's melee attack
ranged nil :RAZORWIND Symbol The internal name of this character's ranged attack
speed nil 8 Integer How fast this character moves
hp nil 30 Integer The maximum HP of this character
melee_damage nil 4 Integer The amount of damage this character's melee attacks deal (can be modified on a move-by-move basis)
ranged_damage nil 4 Integer The amount of damage this character's ranged attacks deal (can be modified on a move-by-move basis)
aim_range [16,9] [10,10] Array[2]<Integer> How far the player can aim for ranged attacks (in tiles) (first value is x-axis, second is y-axis)
aim_type :DEFAULT :EIGHTS Symbol The way the player will aim for ranged attacks
movement_type :DEFAULT :FLYING Symbol This decides what tiles this character can and cannot move on/through
hitbox [0,0,80,80] [20,20,60,60] Array[4]<Integer> The hitbox of this character (x, y, width, height)
guard_time 1 5 Integer How long this character can hold guard until it 'breaks'
guard_cooldown 1 10 Integer How long this character has to wait until it can guard again
unguard_time 1 2 Integer  
dash_distance 0 4 Integer How far this character will dash when guarding (0 means it cant dash)
dash_speed 100 50 Integer How fast this character will dash (this is ignored if the character cant dash)
unlock_proc proc { next true } proc { next $player.level >= 50 } Process A small piece of code that decides what conditions have to be met in order to unlock this character
playable true false Boolean This decides whether the player can play as this character (it shows up in character select)
evolution nil :HONCHKROW Symbol The internal name of the character this evolves into (nil means it doesn't evolve)
evolution_exp 0 60 Integer How much 'match experience' is required to evolve (this is ignored if it doesn't evolve)
description "" "A cool bird Pokémon!" String The description that is shown on the character select screen while hovering over it

 

Name

This only affects the name that is shown on the Character Selection screen

Internal

This value decides a lot of factors and is used for a whole lot of dependancies like:

  1. When another character evolves into this it references this value
  2. When the announcer calls out this character's name it is referencing this value (as will the filename for the announcer Voice Personal N_INTERNALNAME.ogg)
